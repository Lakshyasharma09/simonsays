# Simon Says Game 🎮

A fun memory-based game built using *HTML, CSS, and JavaScript*.

## 🚀 How to Play
- Press any key to start the game.
- Follow the sequence of colors shown.
- Repeat the pattern. Each level gets harder!

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
